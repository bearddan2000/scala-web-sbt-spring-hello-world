# scala-web-sbt-spring-hello-world

## Description
A minimalist build for springframework
on scala.

## Tech stack
- sbt
- scala

## Docker stack
- openjdk:8-jre-alpine

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
